# MTR Network Visualization Project

This project implements a GUI application that performs MTR network analysis and visualizes the results as jets in the sky using Python.
The visualization shows network hops as points in space with latency represented along the path.

## Project Structure
- `src/`: Source code directory
  - `network/`: Network-related functionality
  - `gui/`: GUI components
  - `visualization/`: Visualization components
- `requirements.txt`: Project dependencies
